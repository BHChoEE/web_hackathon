# User-friendly Paper Query System

### Online demo
Now available at https://paperquery.online/

### Usage

1. Get Packages
   ```npm install```

2. Build src
   ```npm run build```

3. Start Server
   ```npm run start```

4. Visit `localhost:3000` at your browser :)
